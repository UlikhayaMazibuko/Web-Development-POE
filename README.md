## Project Overview

This project involves designing and developing a responsive website for **Panto Hair & Beauty (PHB)**, a professional salon offering hair, beauty, and nail services. The site showcases PHB’s brand identity, services, and promotions while providing easy ways for users to engage with the business through booking, reviews, and contact forms.


## Website Goals and Objectives

* Establish a strong digital presence for PHB
* Showcase services, products, and promotions
* Provide accessible booking and contact options
* Support business growth through social media integration


## Key Features and Functionality

* **Homepage:** Branded visuals, promotions, and call-to-action buttons
* **About Us:** Mission, values, and salon story
* **Services & Products:** Detailed descriptions, pricing, and booking links
* **Contact Page:** Business details, social media links, and interactive forms
* **Responsive Design:** Mobile-friendly layouts and typography scaling
* **SEO Optimised:** Meta tags, structured data, image alt text, and internal linking
* **JavaScript Enhancements:** Accordion FAQs, modals, lightbox gallery, interactive map, dynamic content, and form validation


## Technologies Used

* **HTML5, CSS3, JavaScript (ES6+)**
* **Google Fonts:** Nunito and Baloo 2
* **Leaflet.js:** Interactive map
* **CSS Variables:** Typography and spacing system
* **ARIA & Accessibility Features** for keyboard navigation and screen readers


## How to Run

1. Clone or download the repository
2. Open `index.html` in your browser
3. Navigate through the pages using the menu
4. Test forms by entering valid input and submitting
5. Use developer tools to preview responsiveness across devices
